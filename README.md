# Setup

```bash
git clone https://github.com/nguyencong1227/T5_Summarize.git
cd T5_Summarize
pip3 install -r requirements.txt
git clone https://huggingface.co/MBZUAI/LaMini-Flan-T5-783M
```

# Run app
```bash
streamlit run app.py
```
